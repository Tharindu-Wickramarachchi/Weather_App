# Weather


  <img align="left" src="https://github.com/Tharindu-Wickramarachchi/Weather_App/blob/main/assets/logo/App_logo.png" alt="Weather app logo" width="100" style="margin-right: 10px;">
  
 Weather is a weather application, developed using Flutter and powered by the OpenWeather One Call API.
 Stay informed about current weather forecasts with ease. Featuring a sleek interface and precise data,<br>
 WeatherNow ensures you're always up to date with the latest weather conditions.
